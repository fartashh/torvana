# Wildlife Requests (311)

**Service Overview:**
The 311 system handles wildlife slightly differently. Raccoons are considered friendly neighborhood pets and are not rabies vectors.

**Service Level Agreement (SLA):**
- **SLA Timeframe:** Exactly **24 hours** response time for any animal issue.
- Please attempt to safely catch the animal yourself before calling.

**Contact Information:**
- **Wildlife Dispatch:** 555-WILD
- **Email:** wildlife.catch@toronto.fake.ca
